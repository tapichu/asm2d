asm2d

asm2d is an assembler for the 68112D microprocessor. The 68112D is a clone of the 68HC11 microcontroller, with new instructions to generate 2D graphics on a VGA display.

The assembler turns the assembly code into an MIF file that can be used to initialise internal memory (RAM or ROM) during compilation and/or simulation (on an FPGA).

Installation

Using pip:

pip install asm2d

For development

python setup.py develop

Running

asm2d source.s2d

Help

asm2d --help
